# Parkinson's Disease Prediction using CatBoost, Optuna, and SMOTE

This project detects Parkinson's Disease using voice-based biomarkers analyzed with a CatBoost model optimized by Optuna and balanced with SMOTE.  
A Flask-based web app provides real-time, non-invasive PD prediction.
